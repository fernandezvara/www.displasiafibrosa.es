# Imágenes del Sitio Web

Esta carpeta contiene todas las imágenes utilizadas en el sitio web de la Asociación de Displasia Fibrosa.

## Imágenes Requeridas

### libro-mateo.jpg

- **Tamaño**: 500x500 px
- **Formato**: Circular
- **Descripción**: Portada del libro "Un regalo llamado Mateo"
- **Estado**: Necesita añadir la imagen real

### logo.svg

- **Formato**: Vectorial SVG
- **Descripción**: Logo de la Asociación de Displasia Fibrosa
- **Estado**: Necesita crear el logo

## Instrucciones

1. Añade las imágenes reales en esta carpeta
2. Asegúrate de que los nombres coincidan exactamente
3. Las imágenes deben estar optimizadas para web
4. Mantén los formatos y tamaños especificados

## Formatos Soportados

- **JPG**: Para fotografías y imágenes complejas
- **PNG**: Para imágenes con transparencia
- **SVG**: Para logos e iconos vectoriales
- **WEBP**: Para mejor compresión (opcional)
